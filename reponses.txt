REPONSES

1 - Il existe 2730 façons différentes : 15 x 14 x 13 = 2730.

2 - Il existe 32760 façons différentes : 15 x 14 x 13 x 12 = 32760.

3 - Il existe 6 façons de ranger les lettres ABC :
ABC ACB BCA BAC CAB CBA
3 x 2 x 1 = 3! = 6.

4 - Il existe 24 façons de ranger ABCD :
4 x 3 x 2 x 1 = 4! = 24.

5 - Nous avons 120 possibilités:
(10 x 9 x 8) / (3 x 2 x 1) = 10! / (10-3)! / 3! = 120

6 - Nous avons 210 possibilités:
(10 x 9 x 8 x 7) / (4 x 3 x 2 x 1) = 10! / (10-4)! / 4! = 210
